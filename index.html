2. Purpose

ဤ SOP သည် IT Department ၏ လုပ်ငန်းများကို စနစ်တကျ၊ လုံခြုံမှုရှိစွာ၊ စံနှုန်းတူ ဆောင်ရွက်နိုင်ရန် ရည်ရွယ်သည်။

3. Scope

Hardware, Software, Network, Security, User Support, Reporting, Backup, Policy & Compliance အားလုံးအတွက် အကျုံးဝင်သည်။

4. Organization Structure & Responsibility

5. IT Department Policies

5.1 IT Governance Policy

IT Policy နှင့် SOP များအားလုံးကို IT Manager မှ အတည်ပြုပြီးမှ အသုံးပြုရမည်။

5.2 User Support Policy

IT Support ကို Ticket / Email / Phone ဖြင့်သာ လက်ခံရမည်။

5.3 Hardware & Asset Policy

Hardware Asset များကို Asset Register ဖြင့် အပြည့်အစုံ ထိန်းချုပ်ရမည်။

5.4 Software Policy

Licensed Software များသာ အသုံးပြုခွင့်ရှိသည်။ Unauthorized Software မသုံးရ။ (လိုင်စင်နဲ့ မွဲကုန်လိမ့်မယ် ဘယ်သူမှ လိုင်စင်ဗားရှင်းမသုံးဘူး သူခိုးဗားရှင်းပဲထည့်)

5.5 Network & Internet Policy

Unauthorized Network Device မတပ်ဆင်ရ။ Network Change များကို Approval လိုအပ်သည်။

5.6 Information Security Policy

Password Policy, Account Sharing မပြုရ။ Security Incident များကို ချက်ချင်း Report တင်ရမည်။

5.7 Backup & DR Policy

Critical Data များကို Daily Backup ပြုလုပ်ပြီး Restore Test ကို လစဉ် လုပ်ရမည်။

5.8 Reporting Policy

Daily / Weekly / Monthly IT Report မတင်ပါက Non‑Compliance အဖြစ် သတ်မှတ်မည်။

5.9 SOP Compliance Policy

SOP မလိုက်နာပါက Disciplinary Action ခံရနိုင်သည်။

6. Daily Operations SOP

  • User Support & Helpdesk
  • Hardware & Software Management
  • Network & System Monitoring
  • Security & Incident Handling

7. Change & Incident Management

System Change များကို Approval + Rollback Plan ဖြင့်သာ ဆောင်ရွက်ရမည်။ Incident များကို Severity အလိုက် Escalation ပြုလုပ်ရမည်။

8. Reporting & Review

  • Daily Report → IT Leader
  • Weekly Report → Assistant Manager
  • Monthly Report → Management

9. Training & Awareness

IT SOP, Security Awareness Training ကို Staff များအား အနည်းဆုံး နှစ်ละครั้ง ပေးရမည်။
